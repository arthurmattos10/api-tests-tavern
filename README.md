# 🧪 API Tests com Tavern

Projeto de automação de testes de API utilizando Tavern e Pytest.

## 📌 Tecnologias
- Python
- Pytest
- Tavern

## 🚀 O que foi implementado
- Teste de fluxo de tarefas (To-Dos)
- Parametrização de testes
- Encadeamento de dados entre requests
- Validação customizada com Python

## ▶️ Como rodar

```bash
pip install pytest tavern[pytest]
pytest -v
